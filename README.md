# cypress-todomvc-tests

### 1 Create package.json

`npm init -y`

### 2 Install Cypress

`npm install cypress `

### 3 Open Cypress

`npx cypress open `

### Cypress CLI For CI

`npx cypress run `

### Cypress Run specific test

`npx cypress run --spec cypress/integration/todo-mvc-actions.spec.js`

### Scripts Added

`npm run cypress => cypress open`
`npm run test => cypress run`
